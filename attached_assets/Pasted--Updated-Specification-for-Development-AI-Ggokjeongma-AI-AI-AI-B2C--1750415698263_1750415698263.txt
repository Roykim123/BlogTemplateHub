[Updated Specification for Development]

테마: 걱정마AI (Ggokjeongma AI)

걱정마AI는 AI를 통해 모든 작업을 자동화하는 B2C SaaS 기반 플랫폼입니다. 사용자 친화적인 인터페이스와 콘텐츠 중심의 기능을 통해 상업적 활용성에 초점을 둡니다.

0. UI 및 디자인 개선 사항

좌측 메뉴 클릭 시 파란 테두리 박스는 유지하되, 아이콘이 항상 보이고 강조 영역이 명확하게 시각화되도록 개선

아이콘에 마우스를 가져가면 항목명과 간단한 설명이 툴팁 형태로 표시 (REPLIT 스타일 참고)

좌측 상단 로고는 이미지 파일로 교체하되, "AI" 글자는 에르메스 주황색, 굵고 크게 강조

전체 톤앤매너: 중요 UI 요소에 에르메스 주황색 포인트 적용

아이콘은 입체감이 살짝 느껴지도록 그라데이션 혹은 그림자 적용

좌측 아이콘 크기 확대 및 분류 구조 정리:

제일 위: 대시보드

그 아래: 딸깍AI (블로그 자동화)

추가기능: 유튜브, 스레드 등

게시판 기능: 커뮤니티, 외주 게시판 등

마이페이지 영역: 마이페이지 / 결제정보 / 환경설정 등

1. 메인페이지 구성 (튜토리얼 영상 중심)

화면 상단 2/3: 서버 접속 시 자동 재생되는 튜토리얼 영상 표시

하단 1/3: 프리미엄 콘텐츠 영상, 외부 링크 기능 포함

좌측 사용자 게시판에는 커뮤니티 게시판의 최신글을 실시간 불러오기

2. 기능 구조 및 게시판

채팅 기능은 제외

수익화: AI 캐시를 사용하는 미니게임 (사다리, 홀짝 등) - 합법적, 도박성 최소화, 기본 베팅 1캐시 (예: 서공시 2배)

좌측 메뉴에 “외주 게시판” 추가:

자동 개발 외주, 마케팅 외주 요청 가능

글 작성 시 코인 차감 적용

좌측 메뉴에 “커뮤니티” 추가:

자유게시판, 건의사항 투표, 외주게시판 포함

클릭 시 전체화면 페이지로 전환

글 작성/댓글/좋아요/신고 기능 지원

정렬 기준: 최신순, 인기순, 조회수순, 댓글순

게시글 작성 시 1캐시 차감, 건의사항은 무료

3. 도구 목록 구조 고도화 및 목적별 템플릿

기존의 평면 도구 나열방식 폐지

도구 카테고리 탭 구성: 전체, 블로그, 스레드, 유튜브, 인스타그램, 좋아요/댓글 등

각 탭 클릭 시 입력창 및 드롭다운 템플릿 표시 (엑셀 데이터 유효성 검사처럼)

도구목록 아래:

업무 진척도 대시보드 삽입:

키워드 입력 → AI 생성(캐시 차감) → 인터넷 접속 → 업로딩 → 게시완료

각 단계별 색상 변화로 시각적 진행 상황 표시 (애니메이션 없이 단순 UI로 구성)

주요기능: 블로그 자동작성 템플릿

블로그 클릭 시 5개의 아이콘 템플릿 옵션 제공

방문자 늘리기 (기본): 입력 없이 버튼 클릭만으로 발행. 고객설명으로는 황금키워드 기준으로 글을 작성하여 업로드 하여 방문자가 지속적으로 진입되게 합니다.

매장 홍보용 블로그: url링크, 제품/상호명, 주요 키워드, 해시태그 입력 필드 제공. 1개 무료 저장 / 이후 코인 차감

체험 후기, 여행기 등 리뷰형 블로그: SEO 최적화 기반 광고주용 글 자동작성

전문 정보 블로그: 전문가의 글을 참고한 SEO형 2000자 블로그 생성

부업/제품홍보 블로그: 스마트스토어, 쿠팡 링크 기반 글 작성

작성된 글은 미리보기 및 텍스트 수정 가능

"발행하기" 클릭 시 크롬 익스텐션을 통해 네이버 블로그 작성기로 이동

글 하단에 “귀여운이모티콘+ 2코인 사용됨” 안내 표시

자동 저장 기능은 선택 사항

고객 마이페이지: 최근 작업내용 및 캐시 소모 이력 대시보드 제공

SNS 업로드 통합:

“스레드 업로드” 버튼 클릭 시 요약 콘텐츠 자동 생성 (트위터, 인스타 용)

캐시 차감 발생

유튜브 숏츠 생성기 (예정 기능):

요약 및 영상 편집 자동화 기능 포함

“곧 출시” 표시

순위 조회 기능 (추가 검토):

콘텐츠 반응 및 채널 통계 보기용

“곧 출시” 표시

4. 결제 페이지 구성

결제 시스템: 토스 결제 연동 예정 (현재는 디자인만 필요)

월정액 / 연간 요금제 제공, 연간 결제 시 약 10% 할인

요금제:

$6.99 (80 캐시)

$14.99 (200)

$29.99 (450)

$59.99 (1000)

$99.99 (2000)

$219.00 (3500)

연간 결제 버튼 클릭 시 자동 할인 반영, 해제 시 원가 복귀

주요 지표:

매출

AI 캐시 사용량

작성 글 수

활성 유저 수

미니게임 사용량

필터 기능:

날짜별

사용자 등급별

콘텐츠 유형

채널 (블로그, 인스타 등)

데이터 항목:

고객 정보

일별 품목별 매출

GPT API 사용량 및 비용

콘텐츠별 캐시 차감 건수 및 이용기기 정보

엑셀 다운로드 기능 포함 (XLSX)

관리자 대시보드 내 기능 항목 별 드롭다운 및 카드 UI 구성으로 프론트 적용

6. 기술 및 예외 처리 고려사항

최적화된 반응형 웹 대응:

모바일 방문자 80% 수준이므로 최적화 중요

영상 및 콘텐츠는 flex 기반으로 전환 가능하도록 구성

에러 핸들링 / 예외 처리 시나리오:

서버 응답 지연 시 로딩 스피너 제공

발행 실패 시 사용자에게 리트라이 옵션 제공

캐시 부족 시 자동 결제 유도 팝업

네트워크 오류 발생 시 캐시 미차감 및 안내 메시지 출력

7. 회원 등급 및 레벨 시스템

사용자 등급은 무지개색 7단계로 구성:

1단계: Red (가장 낮은 등급)

2단계: Orange

3단계: Yellow

4단계: Green

5단계: Blue

6단계: Indigo

7단계: Violet (최상위)

기준: 누적 캐시 사용량, 콘텐츠 생성 수, 미니게임 참여 횟수 등

등급별 혜택:

월별 무료 캐시 지급

작성 글 자동 요약 횟수 증가

게시글 정렬 상단 우선 노출

외주 게시판 요청 자동 승인 기능

마이페이지 내 “내 등급 현황 및 레벨업 조건” 항목 명시

게시판 참여도 및 좋아요/댓글 반응 등도 레벨포인트에 반영

레벨업 시 텍스트 배지와 축하 메시지만 출력 (애니메이션 생략)
